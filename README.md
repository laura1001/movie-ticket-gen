**功能：** <br>
1.输入电影名后，自动生成正反两张海报 <br>
2.支持更换两张海报的内容与背景颜色 <br>
3.自行输入观影信息，并挑选好合适的颜色与海报内容，点击下载即可获得两张电影海报制成的票根 <br>
4.可以统一下载一批，合并做成正反两面精美的票根 <br>
<br>
注：如生成速度较慢，可能因为API调起缓慢或网络问题，可切换网络后再次进入网站。
